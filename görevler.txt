### ‚òê Lesson 5: Continuous RV's, Mean and Variance (Pages 126-160)
- [ ] **Probability Density Function (PDF)**
    - [ ] Definition (for Continuous RV's)
    - [ ] PDF Properties: $f_X(x) \ge 0$ and $\int_{-\infty}^{\infty} f_X(x) dx = 1$
    - [ ] Probability Calculation: $P(a < X \le b) = \int_{a}^{b} f_X(x) dx$
    - [ ] PDF ‚Üî CDF relationship: $F_X(x) = \int_{-\infty}^{x} f_X(\tau) d\tau$ and $f_X(x) = \frac{dF_X(x)}{dx}$

- [ ] **Expected Value (Mean) and Variance**
    - [ ] Mean (Mean) $E[X] = \mu_X$:
        - [ ] Discrete: $\mu_X = \sum_{k} x_k p_X(x_k)$
        - [ ] Continuous: $\mu_X = \int_{-\infty}^{\infty} x f_X(x) dx$
    - [ ] Moment ($E[X^n]$)
    - [ ] Variance (Variance) $\sigma_X^2 = \text{Var}(X)$:
        - [ ] Definition: $\sigma_X^2 = E[(X - \mu_X)^2]$
        - [ ] Calculation formula: $\sigma_X^2 = E[X^2] - (E[X])^2$
    - [ ] Standard Deviation: $\sigma_X = \sqrt{\text{Var}(X)}$

    - [ ] Joint PMF: $p_{XY}(x_i, y_j) = P(X = x_i, Y = y_j)$

    - [ ] Joint PDF: $f_{XY}(x, y) = \frac{\partial^2 F_{XY}(x, y)}{\partial x \partial y}$

## Chapter 3: Circuit Theorems

    - [ ] **Thevenin:** Model circuit as $V_{TH}$ and $R_{TH}$
    - [ ] **Norton:** Model circuit as $I_{N}$ and $R_{N}$
    - [ ] **Test Source Method:** Apply test voltage or current to find $R_{TH}$ for circuits with dependent sources
- [ ] **Maximum Power Transfer**

## Chapter 5: Capacitors, Inductors and Transient Analysis
*Analysis of time-varying circuits.*

- [ ] **Energy Storage Elements**
    - [ ] **Capacitor:** Current-Voltage relationship ($i = C \frac{dv}{dt}$) and stored energy ($W = \frac{1}{2}CV^2$)
    - [ ] **Inductor:** Voltage-Current relationship ($v = L \frac{di}{dt}$) and stored energy ($W = \frac{1}{2}LI^2$)
    - [ ] Under DC source, Capacitor acts as "Open Circuit", Inductor acts as "Short Circuit"
- [ ] **First Order Circuits (RC and RL)**
    - [ ] **Natural Response:** Energy discharge in sourceless circuit
    - [ ] **Step Response:** Circuit response when DC source is connected
    - [ ] **Time Constant ($\tau$):** For RC circuit $\tau = RC$, for RL circuit $\tau = L/R$
    - [ ] **General Solution Formula:** $x(t) = x(\infty) + [x(0) - x(\infty)]e^{-t/\tau}$ formula

        - [ ] Underdamped / Oscillatory: $\alpha < \omega_0$
